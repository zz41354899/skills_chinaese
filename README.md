# Agent Skills 中文版

這是一個使用 Vue 3 + Vite + TypeScript + Tailwind CSS 建構的 Agent Skills 中文版網站。

## 功能特色

- 🎨 現代化的 UI 設計
- 📱 完全響應式佈局
- 🌙 深色主題
- ⚡ 使用 Vite 實現快速開發
- 🔧 TypeScript 支援
- 🎯 Vue 3 Composition API

## 技術棧

- **框架**: Vue 3
- **建構工具**: Vite
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **路由**: Vue Router

## 開始使用

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

### 建構生產版本

```bash
npm run build
```

### 預覽生產版本

```bash
npm run preview
```

## 專案結構

```
src/
├── components/       # 可重複使用的組件
│   └── NavBar.vue   # 導航列組件
├── pages/           # 頁面組件
│   ├── HomePage.vue
│   ├── WhatAreSkillsPage.vue
│   └── SpecificationPage.vue
├── App.vue          # 根組件
├── main.ts          # 應用程式入口
└── style.css        # 全域樣式
```

## 頁面說明

- **首頁 (/home)**: Agent Skills 概覽和主要功能介紹
- **什麼是技能? (/what-are-skills)**: 詳細說明技能的概念和組成
- **規格說明 (/specification)**: 技術規格和範例
